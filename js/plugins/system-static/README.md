# @parel/system-static

> PAREL plugin for static system prompt injection.

A first-party runtime plugin for [PAREL](https://github.com/parall-hq/parel-oss).

## Install

```bash
npm install @parel/system-static
```

## Usage

```yaml
plugins:
  - plugin: system-static
    config:
      prompt: You are a concise assistant.
```
