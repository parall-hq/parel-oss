# @parel/steering-immediate

> PAREL plugin for immediate steering and interruption inputs.

A first-party runtime plugin for [PAREL](https://github.com/parall-hq/parel-oss).

## Install

```bash
npm install @parel/steering-immediate
```

## Usage

```yaml
plugins:
  - steering-immediate
```
