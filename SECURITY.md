# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| latest  | Yes       |

## Reporting a Vulnerability

**Please do not open a public issue for security vulnerabilities.**

Report vulnerabilities via [GitHub Private Vulnerability Reporting](https://github.com/parall-hq/parel-oss/security/advisories/new) or email **security@parel.sh**.

We will:

1. Acknowledge receipt within 48 hours.
2. Provide an initial assessment within 7 days.
3. Work with you on a fix timeline and coordinate disclosure.

## Scope

This policy covers all code in this repository, including:

- SDK packages (`@parel/core`, `@parel/plugin-sdk`)
- First-party plugins (`@parel/sandbox-*`, `@parel/security-*`, etc.)
- CLI (`@parel/cli`)
- JSON schemas and protocol definitions
